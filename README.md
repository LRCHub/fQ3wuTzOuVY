錆びて朽ちた 鍵束でも
希望 一縷抱けるなら
守るべき宝物
愛しさが罪過に変わる前に
嘆きの扉を こじ開けて進め
誰ガ為ノ世界だとしても
強く儚く優しい 真実と嘘と
裏切りと罪と その全て受け止めて
哀しき運命を 生きる者よ
滅びの振り子に あらがう者よ
いつかいつの日か 仰ぐ天に
やすらぎの音が 響くように－

誰ガ為ノ世界だとしても
遥か遠き日の夢の輪郭が今も
鮮やかなままで
ただ此処にあるから
